$$\sin ( x \mp y ) = \sin x \ \cos y \mp \cos x \ \sin y$$

$$\sin x \sin y = \dfrac{1}{2} \left[ \cos( x - y ) - \cos ( x + y ) \right]$$
$$\sin x \cos y = \dfrac{1}{2} \left[ \sin( x - y ) + \sin ( x + y ) \right]$$
